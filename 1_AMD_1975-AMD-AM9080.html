AMD AM9080

Histoire:

Am9080 est un processeur fabriqué par AMD.
Il a été initialement produit sans licence comme un clone de l'intel 8080
En 1976, un accord a été conclu avec Intel pour le fabriquer. Les premieres versions de ce 
processeur on été disponible en avril 1974.

Description:

  • Clock Speed : 2Mhz - 4Mhz
  • Nbr de Core : 1
  • Architecture : 8 bit
  • Transmitor Count : 4.500
  • Fabrication Process : 6.000nm
